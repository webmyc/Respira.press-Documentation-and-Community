---

Cursor users have two paths: connect the Respira MCP server through Cursor's built-in MCP client, or call `respira` directly from the integrated terminal.

## Using the integrated terminal

```bash
respira read structure mysite.com
respira read design-system mysite.com
```

## Using the MCP server

Point Cursor's MCP client at `@respira/wordpress-mcp-server` for richer tool access. See the [MCP server docs](/mcp-server) for setup.
