Read content from connected WordPress sites. Two commands work on any public site without authentication.

## Public, no auth required

```bash
respira read structure <url>
respira read design-system <url>
```

These commands parse the public HTML for builder signatures, global colors, fonts, and a best-effort sitemap. No account, no API key.

## Authenticated, connected sites

```bash
respira read page <site> <page> [--as=builder|html|portable]
respira read pages <site> [--builder=elementor]
respira read post <site> <post>
respira read posts <site> [--type=product --limit=50]
respira read media <site> [--type=image]
respira read taxonomy <site> <taxonomy>
```

### --as format options

- `builder` (default): builder-native JSON. Preserves every attribute and id.
- `html`: rendered HTML as WordPress would emit it.
- `portable`: flattened Portable Text for migrations.
