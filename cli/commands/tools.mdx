---
title: respira tools
description: Browse the Respira tool catalog from the CLI. List, describe, or search across all available tools.
---

Every Respira tool is addressable from the CLI and the SDK.

```bash
respira tools list [--for=elementor|task:content-editing|site:mysite.com]
respira tools describe <tool-name>
respira tools search "update typography"
```

`--for` accepts builder names, task categories, and site references. The same catalog is exposed to Claude Code, Cursor, and other MCP clients.
