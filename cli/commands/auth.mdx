---
title: respira auth
description: Reference for respira auth login, logout, status, and whoami.
---

Authenticate the CLI, check status, or sign out.

## login

```bash
respira auth login [--no-browser]
```

Opens a browser, performs the handshake with respira.press, and stores the API key in your OS keychain. Use `--no-browser` to print the URL instead.

## logout

```bash
respira auth logout
```

Removes the stored key from the keychain and any fallback file.

## status

```bash
respira auth status
```

Shows whether you're authenticated, your plan tier, and remaining rate-limit quota.

## whoami

```bash
respira auth whoami
```

Prints the authenticated account's email and workspace.
