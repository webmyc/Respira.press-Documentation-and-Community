title: respira docs
description: Read Respira builder documentation from the terminal. Ships with embedded v1 docs for Elementor, Divi, and Bricks.
---

Look up builder documentation without leaving your terminal.

```bash
respira docs elementor
respira docs elementor widget-typography
respira docs divi shortcode-syntax
respira docs --search="dynamic tag"
```

v1 ships with embedded docs for Elementor, Divi, and Bricks. Later versions fetch from the Respira docs API for always-current content.
